Deep Java Library(DJL)的。DJL是亚马逊在2019年宣布推出的开源的深度学习开发包，它是在现有深度学习框架基础上使用原生Java概念构建的开发库。DJL不依赖于具体的引擎和深度学习框架，可以随时切换框架。它是为Java开发者量身订造的开发工具。
