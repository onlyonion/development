## 经典理论
* 经典风格 管道过滤器、面向对象、事件驱动、层次架构、仓库、解释器、反馈控制
* 风格 数据流风格、调用返回风格、独立构件风格、虚拟机风格、仓库、CS、BS、三层
* 描述 UML、4+1视图、ADL
  * 逻辑视图（功能需求）、开发视图、过程视图（非功能性需求）、物理视图、场景视图评估
* 评估 SAAM基于场景的评估、ATAM体系结构权衡分析方法
