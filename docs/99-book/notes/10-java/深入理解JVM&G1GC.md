### 2.3 Garbage Collection
#### 2.3.7 Garbage First（G1）GC
G1切分堆内存为多个区间（Region），从而避免了很多GC操作在整个Java堆或者整个年代进行。

G1的年轻代收集阶段是一个并行的独占收集器。和其他HotSpot垃圾收集器一样，当一个年轻代收集进行时，
整个年轻代会被回收，所有的应用程序线程**会被中断**。G1GC会启用多线程执行年轻代回收。
