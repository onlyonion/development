### oncd 
进线调异同死，内分段页虚局，  
文目权层调树，控符块缓分回。    
物数网传会表，运控存指线流，  
线栈队表树图，查找内外排序。  

分取递区两边走，分取递合三指针。

### 树
二叉排序红黑树，  
每根叶红路数同，  
多路排序文目库，  
阶根关子序指层。  

### Service
函数调用、系统调用、方法调用、IPC、RPC

注扩启停调，集目路衡虑，  
超重幂限降，隔断流跟压，  
安监报可告，配传编程序。  

### Cache
字链哈希，有序集合，读写分离，主从复制，  
持久事件，哨兵集群，发布订阅，事务排序。

### Message Queue
消息传递 PMSSS 

生消配注监中，主分副日段索，  
偏事延重幂死，用靠同伸路衡。   
更致透击雪过，用丢重积致序。

### Database
事关页排主公，分片键算策略，  
解优路由改写，执行结果合并。  

事务、关联、分页、排序、主键、公共表（全局表、广播表）
